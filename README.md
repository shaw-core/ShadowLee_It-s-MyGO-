# ShadowLee: It's MyGO!

> 一款以虚拟主播 **李豆沙** 及其相关角色设定为基础制作的非官方粉丝同人游戏。
> 在不同世界与叙事图层之间穿行，收集散落的信息，抵达故事表面之下的真相。

## 游戏简介

《ShadowLee: It's MyGO!》是一款融合了：

* 2D 平台跳跃
* 3D 场景探索
* 视觉小说式剧情
* 多图层切换
* 文档收集与密码解谜
* 隐藏路线与特殊结局

的浏览器同人游戏。

玩家将控制李豆沙的不同形象，在看似彼此独立的关卡和剧情片段中寻找线索。

部分信息不会被直接说明。关卡中的草稿页、碎片、系统档案、角色对话以及异常演出，可能共同指向另一个版本的故事。

---

## 游戏内容

### 前编：2D 次元冲刺

在现实图层与漫画图层之间切换。

两个图层具有不同的平台、障碍和通行路线。玩家需要利用图层差异完成平台跳跃，并收集散落在关卡中的草稿页和其他隐藏内容。

### 后编：3D 世界探索

进入三维场景，选择不同的李豆沙形象进行探索。

不同角色外观不仅是视觉差异。某些角色可能拥有与世界系统不同的关系，并能够触发普通角色无法触发的内容。

### 剧情与图鉴

完成关卡、取得特定收集品后，可以解锁角色对话和剧情事件。

已解锁的剧情可以在图鉴中重新查看。

### 隐藏系统

游戏中存在需要额外条件才能访问的系统内容，包括：

* 散落的草稿页
* 加密档案
* 密码终端
* 特殊 CG
* 隐藏角色外观
* 特殊终端演出
* 隐藏结局

部分谜题需要结合多个关卡中的信息才能理解。

## 基本操作

### 2D 模式

| 操作        | 按键                  |
| --------- | ------------------- |
| 向左移动      | `A` / `←`           |
| 向右移动      | `D` / `→`           |
| 跳跃        | `W` / `↑` / `Space` |
| 切换现实与漫画图层 | `Q`                 |
| 返回主菜单     | `Esc`               |

其他模式的操作提示会显示在游戏界面中。

## 推荐游玩方式

为了获得较完整的体验，推荐：

1. 先正常完成前编与后编。
2. 尽量收集每个关卡中的草稿页与碎片。
3. 查看图鉴中解锁的剧情。
4. 留意文档里的日期、编号、措辞和互相矛盾的信息。
5. 尝试使用不同角色重新进入部分内容。
6. 在达成隐藏路线后，再次观察标题画面。

本作的一些关键叙事并不发生在传统剧情对话中，而是隐藏在界面变化、系统反馈和游戏规则本身之中。

## 本地运行

### 环境要求

* Node.js 18 或更高版本
* npm

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

### 本地预览构建结果

```bash

```
